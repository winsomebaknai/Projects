This project demonstrates a real-time data warehouse using MongoDB, PyMongo, Kafka (for streaming), Pandas (for batch processing), and Dash (for visualization). 
It integrates streaming and batch data, processes real-time analytics, and visualizes insights via an interactive web dashboard.


Project Structure
 Step 1: Generate mock data
 Step 2: Insert data into MongoDB
 Step 3: Query & process data
 Step 4: Visualization (Flask/Dash)

Queries Implemented

1- Transaction Analysis: Total spending per category, average transaction per user.
2 -System Performance Analytics: Avg CPU & RAM usage per device, high CPU usage alerts.
3 -User Activity Analytics: Top visited websites, most used browsers.
